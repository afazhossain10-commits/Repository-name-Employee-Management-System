# Employee Management System

## Overview

A Java-based system to manage employee records using ArrayList and HashMap with CRUD operations and file handling.

## Features

* Add, View, Update, Delete employees
* Search by name
* File save/load
* Salary report generation

## Run

javac *.java
java EmployeeManagementSystem

## Structure

* src/ → source code
* data/ → employee file
* docs/ → documentation
* reports/ → sample report
